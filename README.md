# PS Study

PS(Problem Solving) 공부에 쓰고 있는 저장소입니다.
주로 [백준](https://www.acmicpc.net/)을 풀며, 풀이한 코드를 올리고 있습니다.
일부 코드는 관련 해설을 [개인 블로그](https://blog.lapis0875.com/)에 함께 올리고 있습니다.

## /boj
직접 푼 백준 문제의 코드를 올리고 있습니다. 
푼 문제 수가 많아짐에 따라, 1000 단위로 문제 번호를 나누어 폴더로 묶어 관리하고 있습니다.

## /data_structure
자료구조에 관한 기본적인 내용을 직접 구현해보며 공부합니다.
시간 날 때 하나씩 구현해보려 합니다.

## /boj_validator
문제풀이에 쓰고 있는 간단한 도구입니다. 
테스트 케이스 채점, 문제 풀이를 저장할 경로 생성, 폴더 구조 정리 등의 기능을 제공합니다.
추후 이름 변경 및 재작성 예정입니다.
